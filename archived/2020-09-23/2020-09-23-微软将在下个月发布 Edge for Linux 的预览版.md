[微软将在下个月发布 Edge for Linux 的预览版](https://www.solidot.org/story?sid=65620)
------

<div>
微软 在  Ignite 2020 上<a href="https://venturebeat.com/2020/09/22/microsoft-edge-linux-developers-october-2020/">宣布</a>将在下个月发布 <a href="https://www.microsoftedgeinsider.com/" target="_blank">Edge for Linux</a> 的预览版本。微软基于 Chromium 的新 Edge 浏览器早在今年初就发布了支持 Windows 7、Windows 8、Windows 10 和 macOS 版本，它目前是仅次于 Chrome 的第二大浏览器。Edge 支持 Linux 并非是为了扩大在浏览器市场的份额，而是为了更好的支持企业级用户，为企业用户提供单一的浏览器部署方案，可以部署到所有雇员机器上，不管运行的是 Linux、Windows 还是 macOS。 Edge for Linux 的预览版将首先支持 Ubuntu 和 Debian 发行版。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
